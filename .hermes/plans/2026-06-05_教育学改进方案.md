# 三年级语文复习App — 教育学改进方案

## 当前问题分析

现有版本功能完整但偏"静态"：复习清单是纯阅读，测试是做完一次性评分。
缺乏：追踪薄弱点、间隔复习、主动回忆、个性化路径。

---

## 改进方案（基于7个教育学理论）

### 1. 间隔重复系统（Spaced Repetition）
**理论依据**：Ebbinghaus遗忘曲线 + SuperMemo算法
**做什么**：
- 记录每道题的对错历史
- 答错的题目自动加入"错题本"
- 按遗忘曲线安排复习：错1次→1天后复习，再错→3天后，再错→7天
- 首页显示"今日待复习"提醒

**实现**：localStorage存答题记录，前端计算下次复习时间

### 2. 检索练习（Retrieval Practice / Testing Effect）
**理论依据**：Roediger & Karpicke (2006) — 测试比重复阅读效果好2倍
**做什么**：
- 复习清单改为"闪卡模式"：先遮住答案，让孩子主动回忆
- 古诗：只显示标题，孩子尝试背诵，再点击揭晓
- 字词：显示拼音，孩子先写汉字，再对照
- 重点句子：挖空关键部分，先填再看

**实现**：CSS遮罩 + 点击翻转动画

### 3. 交错练习（Interleaving）
**理论依据**：Rohrer & Taylor (2007) — 混合练习比分类练习效果好
**做什么**：
- 测试题不再按"选择/填空/阅读"分块
- 打乱顺序随机出题
- 同一题型内混合不同知识点
- 每次测试随机抽取20题（从8单元题库中）

### 4. 脚手架式学习（Scaffolding / ZPD）
**理论依据**：Vygotsky最近发展区
**做什么**：
- 难度分级：⭐基础 → ⭐⭐进阶 → ⭐⭐⭐挑战
- 基础：直接填空（给拼音写汉字）
- 进阶：语境填空（给句子挖空）
- 挑战：无提示默写
- 答对基础才能解锁进阶

**实现**：解锁机制 + 难度标签

### 5. 形成性评价 + 即时反馈（Formative Assessment）
**理论依据**：Black & Wiliam (1998) — 即时反馈比延迟评分效果好
**做什么**：
- 每答一题立即显示对错 + 解析
- 不再等"提交评分"才出结果
- 答错时显示：正确答案 + 为什么对 + 相关知识点链接
- 错题自动归类（字词/古诗/阅读/理解）

**实现**：onclick即时反馈 + 解析弹窗

### 6. 游戏化激励（Gamification）
**理论依据**：Deci & Ryan 自我决定理论（自主+胜任+归属）
**做什么**：
- 积分系统：答对+10分，连续答对+5分bonus
- 等级系统：识字新秀→语文小能手→古诗达人→语文学霸
- 连续学习天数统计（streak）
- 成就徽章：🏆首次满分、🔥连续7天、📚背完所有古诗
- 排行榜（本地，多个孩子用同一设备时）

**实现**：localStorage + 动画效果

### 7. 多模态学习（Multimedia Learning）
**理论依据**：Mayer多媒体学习原则
**做什么**：
- 古诗配插图（用AI生成水墨风格插画）
- 生字配笔顺动画（SVG/Canvas）
- 重点句子配朗读（Web Speech API）
- 图文结合的记忆宫殿

**实现**：Web Speech API朗读 + SVG笔顺 + CSS动画

## 技术方案

- 全部前端实现（HTML/CSS/JS），无需后端
- 数据存储：localStorage（浏览器本地）
- 语音：Web Speech API（免费，浏览器内置）
- 笔顺：SVG动画或Canvas绘制
- 部署：继续用GitHub Pages

## 改进后的用户体验流程

```
打开App → 看到"今日待复习"（间隔重复）
       → 选择模式：📚复习 / 📝测试 / 🃏闪卡
       
复习模式：翻转闪卡，主动回忆 → 系统记录掌握度
测试模式：随机20题，答一题判一题 → 错题自动入错题本
闪卡模式：古诗/字词/句子 → 点击翻转看答案
       
结束后：看到积分变化 + 等级进度 + 错题统计
```
